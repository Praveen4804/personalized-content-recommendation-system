# Personalized Content Recommendation System

A Netflix-style personalized content recommendation platform built using Flask.

## Features

- 🎬 Movie recommendation system
- 🎵 Music discovery with Spotify links
- 🎮 Embedded browser games
- 👤 User login and signup
- 📊 User watch history tracking
- 🔍 Live search functionality
- 🎥 Movie detail pages

## Technologies Used

- Python
- Flask
- Pandas
- HTML
- CSS
- JavaScript
